# Path Asciifier

## Description

Convert directory or file names have only ASCII characters. Written with TDD methodology in mind.

## Requirements

- Python >= 3.6

## Running tests

```sh
python3 -m unittest
```

## Running the app

```sh
python3 src/main.py
```
